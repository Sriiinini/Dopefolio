Project Showcase with Ministry of Health (Singapore)

This project serves as a comprehensive showcase of my work experience in Singapore COVID Operations. During my tenure, I was deeply involved in monitoring and analyzing the pandemic's progression, providing critical data insights that supported strategic decision-making. To replicate and extend my efforts, I have developed an interactive COVID Dashboard using Power BI, focusing on the analysis of COVID-19 cases and deaths in Singapore.

See Live Dashboard

Project Overview

The primary objective of this project is to illustrate the impact of COVID-19 in Singapore from 2020 to 2021 by providing real-time data visualizations and insightful analysis. This dashboard aims to present key metrics, trends, and patterns related to the pandemic, facilitating better understanding and decision-making for health officials and the general public.

Data Analysis:

Trend Analysis: Analyzed daily, weekly, and monthly trends in COVID-19 cases and deaths to identify patterns and spikes in infection rates.

Geospatial Analysis: Leveraged Google's Geolocation API to map the distribution of cases across different regions in Singapore. This geospatial perspective provided insights into hotspots and the geographic spread of the virus.

Visualization and Dashboard Development:

Power BI: Developed an interactive dashboard that includes various visualizations such as line charts, bar graphs, heat maps, and geospatial maps. These visualizations enable users to explore the data dynamically and gain deeper insights.

User Experience: Focused on creating a user-friendly interface that allows for easy navigation and interaction with the data. The dashboard includes filters and drill-down capabilities to facilitate customized views and analyses.
